# 字体生成器

一个可直接打开，也可以部署到 GitHub Pages 在线分享的字体预览网页：输入中文文字后，实时预览楷书、行楷、行书、草书四种风格，默认展示楷书和行楷。

## 功能

- 输入任意中文内容后即时预览
- 默认自动显示楷书与行楷
- 字体按钮支持二次点击取消选中
- 支持调整字号与行距
- 手机和电脑都做了响应式适配
- 直接打开 `index.html` 就能使用
- 发布到 GitHub Pages 后可直接发链接给朋友

## 字体方案

- 楷书：优先使用热门开源网页字体 `LXGW WenKai / 霞鹜文楷`，再回退到系统楷体
- 行楷：不是直接换现成字体，而是基于楷体做了网页规则优化版，尽量贴近硬笔字帖行楷的结构和连笔节奏
- 行书：使用更顺手的网页字体 `云峰静龙行书`，回退 `Long Cang`
- 草书：优先系统手写风字体，如 `HanziPen SC`，再回退到系统行楷

楷书、行楷、行书都优先使用网页字体，所以在线分享时不同设备看到的效果会更一致；草书仍优先使用系统手写风字体。

## 本地使用

直接双击 `index.html`，或在项目目录运行：

```bash
python3 -m http.server 3810
```

然后打开 `http://127.0.0.1:3810/`。
