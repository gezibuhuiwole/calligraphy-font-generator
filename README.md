# 字体生成器

一个可直接打开的静态网页小工具：输入中文文字后，实时预览楷书、行楷、行书、草书四种风格，默认展示楷书和行楷。

## 功能

- 输入中文内容后即时预览
- 默认自动显示楷书与行楷
- 字体按钮支持二次点击取消选中
- 支持调整字号与行距
- 手机和电脑都做了响应式适配
- 直接打开 `index.html` 就能使用

## 字体方案

- 楷书：优先系统楷体（接近 Word 的 Kaiti / KaiTi），缺失时回退到 `Noto Serif SC`
- 行楷：优先系统行楷，缺失时回退到 `Ma Shan Zheng`
- 行书：`Zhi Mang Xing`，回退 `Long Cang`
- 草书：`Liu Jian Mao Cao`

网页字体来自 Google Fonts，系统字体则按设备已有字体优先显示。

## 本地使用

直接双击 `index.html`，或在项目目录运行：

```bash
python3 -m http.server 3810
```

然后打开 `http://127.0.0.1:3810/`。
